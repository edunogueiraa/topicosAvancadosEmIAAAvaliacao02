## Criação de um ambiente virtual:


### Criação da venv
```bash
python -m venv .venv
```

### Ativação do ambiente
```bash
source .venv/bin/activate
```
### Desativação do ambiente:
```bash
deactivate
```

## Instalar dependências

```bash
pip install -r requirements.txt
```
